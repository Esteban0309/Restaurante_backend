## 📝 Descripción

Este proyecto es la migración y unificación completa de una arquitectura híbrida se encuentra centralizada de manera eficiente en una base de datos relacional utilizando el ORM de Django, ofreciendo respuestas JSON estandarizadas y seguridad por tokens criptográficos.

## 🛠️ Configuración del Proyecto

### 1. Clonar el repositorio e instalar dependencias
Asegúrate de tener Python 3.10+ instalado. Luego ejecuta los siguientes comandos en tu terminal:

```bash
# Crear el entorno virtual
$ python -m venv venv

# Activar el entorno virtual (Windows)
$ venv\Scripts\activate

# Activar el entorno virtual (Mac/Linux)
$ source venv/bin/activate

# Instalar las dependencias requeridas
$ pip install -r requirements.txt
